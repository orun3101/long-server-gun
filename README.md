## 설치 방법 (친구들용)
1. 이 저장소 우측 상단 **Code → Download ZIP** 클릭
2. 압축 풀기
3. `install.bat` 더블클릭 실행
4. 설치 끝나면 마인크래프트 런처에서 **forge 1.20.1** 프로필로 실행
5. 멀티플레이 → 서버 추가 → `192.168.219.108:25567` 입력 후 접속
